# reactjs-application

ReactJS application with WebPack.
Triangle Resolver



## Getting Started
```sh
git clone git@github.com:asmab/reactjs-application.git

cd reactjs-application

npm install

npm start
```

Open [http://localhost:3000](http://localhost:3000)<br>


## Libraries used
* ReactJs
* Webpack
* Less
* React Router 4
* Bootstrap 4
* lodash
* Font Awesome
